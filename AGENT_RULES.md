# Agent Rules and behavior

1. **Server Management**: DO NOT run `uv run main.py` or start the application server instance. The user is running this process manually in their own terminal.
2. **Port Handling**: Assume the server is running on localhost:8080 (or the configured port) without needing to start it yourself.
3. **Git Commit Messages**: Once the changes are completed, provide a commit message that describes the changes in a concise manner.
